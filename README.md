# Hack the North - 2021
https://skilled-creek-301921.ue.r.appspot.com/

## Installation
```bash
$ python3 -m venv .venv
$ source .venv/bin/activate
$ pip install -r requirements.txt
```

## Start
```bash
$ python main.py
```

## Deployment
```bash
$ gcloud app deploy
```

## Logs
``` bash
$ gcloud app logs tail -s default
```
